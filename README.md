# Volume Trading Helper

Небольшое CLI-приложение на Python, которое **в образовательных целях** описывает,
как можно анализировать рынок через объёмы.

## Что внутри

- 3 базовых сетапа по объёмам;
- условия входа/выхода;
- условия отмены сценария;
- блок управления риском.

## Запуск

```bash
python3 app.py
```

## Важно

Материалы в приложении не являются финансовой рекомендацией.
